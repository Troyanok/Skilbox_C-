# Симуляция мобильного телефона

## Описание задания
Реализовать программу на C++, имитирующую работу простого мобильного телефона. Пользователь взаимодействует с программой через консольные команды: `add` (добавить контакт), `call` (позвонить), `sms` (отправить сообщение) и `exit` (выход). Все номера телефонов задаются в формате `+7` и ровно 10 цифр (например, `+71234567890`). Телефон и отдельные записи адресной книги должны быть представлены классами.

## Требования

### Класс `Contact`
Представляет отдельную запись в адресной книге. Содержит:
- `name` — имя контакта (`std::string`)
- `phone` — номер телефона (`std::string`)

Конструктор класса должен инициализировать оба поля. Рекомендуется также предоставить конструктор по умолчанию для удобства.

### Класс `PhoneSim`
Инкапсулирует логику телефона. Должен содержать:
- Контейнер для хранения контактов (например, `std::vector<Contact>`)

Методы, соответствующие командам:
- `AddPhone(const std::string& phone, const std::string& name)` — добавляет контакт в адресную книгу. Предполагается, что номер телефона уже проверен на соответствие формату `+7\d{10}` перед вызовом метода.
- `CallPhone(const std::string& input)` — совершает звонок. Ищет контакт по номеру телефона или по имени. Если контакт найден, выводит в консоль `CALL <номер_телефона>`. Если контакт не найден, выводит сообщение об ошибке.
- `WriteSms(const std::string& input)` — отправляет СМС. Аналогично ищет контакт, запрашивает у пользователя текст сообщения (через стандартный ввод) и выводит информацию о получателе и текст сообщения. При отсутствии контакта сообщает об ошибке.

Внутренняя механика:
- Поиск контакта по строке (номеру или имени) должен быть реализован в отдельном вспомогательном методе (например, `FindPhoneFromList`). Он возвращает признак успеха и сохраняет найденный контакт для последующего использования методами `CallPhone` и `WriteSms`.
- Все операции ввода-вывода, связанные с интерфейсом (кроме запроса текста СМС и вывода сообщений `CALL`/информации о СМС), рекомендуется вынести в основную программу, чтобы класс отвечал только за логику телефона.
- Команда `exit` обрабатывается в `main` и не требует метода в классе.
