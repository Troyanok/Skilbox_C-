# Управление окном рабочего стола

## Описание задания
Реализовать программу на C++, симулирующую поведение окна на виртуальном рабочем столе. Экран монитора имеет фиксированный размер 80×50 пикселей. Пользователь управляет окном с помощью консольных команд: `move` (переместить), `resize` (изменить размер), `display` (показать экран) и `close` (закрыть программу). Координаты окна не могут выходить за границы экрана, ширина и высота должны быть положительными.

## Требования

### Класс `Window`
Представляет окно приложения. Содержит:
- `x`, `y` — координаты левого верхнего угла (`int`)
- `width`, `height` — ширина и высота окна (`int`)

Конструкторы:
- По умолчанию (создаёт окно с нулевыми размерами в точке (0,0))
- С параметрами `(int x, int y, int width, int height)` – должен проверять корректность переданных значений (неотрицательные координаты, положительные размеры, вписывание в экран)

Методы:
- `bool MoveWindow(int dx, int dy)` — смещает окно на вектор (dx, dy). Возвращает `true`, если новое положение полностью внутри экрана, иначе `false` и окно не двигается.
- `bool ResizeWindow(int newWidth, int newHeight)` — изменяет размер окна. Возвращает `true`, если новые размеры положительны и окно остаётся в границах экрана, иначе `false`.
- `int GetX() const`, `int GetY() const`, `int GetWidth() const`, `int GetHeight() const` — геттеры координат и размеров.

Внутренняя механика:
- Вспомогательный метод `CheckInsideScreen(int x, int y, int w, int h)` проверяет, что прямоугольник с заданными параметрами целиком лежит в области `[0, 80) × [0, 50)`.

### Класс `Screen`
Представляет виртуальный монитор. Должен содержать:
- Объект класса `Window` (текущее окно)
- Константы `SCREEN_WIDTH = 80`, `SCREEN_HEIGHT = 50`

Методы:
- `void PrintDisplay()` — выводит текущее состояние экрана в консоль: для каждого пикселя выводится `'1'`, если он принадлежит окну, иначе `'0'`. Вывод построчный, 50 строк по 80 символов.
- Обработка пользовательских команд (рекомендуется вынести в основную программу или отдельный метод `WorkCommand`).

Основной цикл программы (в `main` или отдельном методе):
- Запрашивает начальные параметры окна (координаты и размер) с проверкой корректности.
- Принимает команды:
  - `move` – ввод вектора (dx dy), вызов `Window::MoveWindow()`, вывод обновлённых координат или сообщения об ошибке.
  - `resize` – ввод новых ширины и высоты, вызов `Window::ResizeWindow()`, вывод нового размера или ошибки.
  - `display` – вызов `Screen::PrintDisplay()`.
  - `close` – завершение программы.
