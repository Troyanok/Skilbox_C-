# Имитация аудиоплеера

## Описание задания
Реализовать программу на C++, имитирующую работу простого аудиоплеера, с использованием классов. Программа должна обрабатывать команды пользователя для управления воспроизведением треков.

## Требования

### Класс `Track`
Представляет отдельную аудиозапись. Содержит поля:
- `title` (название, `std::string`)
- `creationDate` (дата создания, `std::tm`)
- `duration` (продолжительность в секундах, `int`)

Конструктор должен инициализировать все поля. Возможно перегрузка оператора вывода для удобного отображения информации.

### Класс `AudioPlayer`
Инкапсулирует логику плеера. Должен содержать:
- Контейнер доступных треков (`std::vector<Track>`)

Методы, соответствующие командам:
- `play(const std::string& trackName)` – начать воспроизведение трека по названию. Если уже что-то играет, команда игнорируется.
- `pause()` – поставить текущий трек на паузу. Если ничего не играет или уже на паузе – ничего не происходит.
- `next()` – переключиться на случайный трек из списка (shuffle). Начинает воспроизведение выбранного трека, останавливая предыдущее, если оно было.
- `stop()` – полностью остановить воспроизведение. Сброс всех флагов, информация о треке не сохраняется.
- `exit()` – завершить программу.
