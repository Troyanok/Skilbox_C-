# Симулятор работы компьютера

Многомодульный учебный проект на C++, моделирующий взаимодействие основных компонентов персонального компьютера.  
Программа позволяет через консольные команды управлять виртуальными процессором, оперативной памятью, диском, видеокартой и клавиатурой.

## Компоненты
```bash
ComputerSimulator/
├── include/
│   ├── cpu.h       # интерфейс центрального процессора
│   ├── ram.h       # интерфейс оперативной памяти
│   ├── disk.h      # интерфейс жёсткого диска
│   ├── gpu.h       # интерфейс графической карты
│   └── kbd.h       # интерфейс клавиатуры
│   ├── constants.h # константные значения
├── src/
│   ├── cpu.cpp    # реализация вычисления суммы
│   ├── ram.cpp    # реализация буфера и операций чтения/записи
│   ├── disk.cpp   # реализация сохранения/загрузки в файл
│   ├── gpu.cpp    # реализация вывода на экран
│   └── kbd.cpp    # реализация ввода с клавиатуры
├── main.cpp       # основной цикл обработки команд
├── CMakeLists.txt
└── README.md
```

### Центральный процессор (cpu)
- `compute()` – считывает 8 целых чисел из оперативной памяти, вычисляет и выводит в консоль их сумму.

### Оперативная память (ram)
- Внутренний буфер на 8 целых чисел.
- `write(index, value)` – записывает число по указанному индексу.
- `read(index)` – возвращает число из буфера.

### Жёсткий диск (disk)
- `save()` – сохраняет 8 чисел из оперативной памяти в файл `data.txt`.
- `load()` – загружает 8 чисел из файла `data.txt` в оперативную память.

### Графическая карта (gpu)
- `display()` – выводит в консоль все 8 чисел из оперативной памяти.

### Клавиатура (kbd)
- `input()` – запрашивает у пользователя 8 целых чисел и записывает их в оперативную память.

## Команды управления

| Команда    | Действие |
|------------|----------|
| `sum`      | Вычислить и вывести сумму 8 чисел из ОЗУ (используется CPU) |
| `save`     | Сохранить данные из ОЗУ на диск (файл `data.txt`) |
| `load`     | Загрузить данные с диска в ОЗУ |
| `input`    | Ввести 8 чисел с клавиатуры и поместить их в ОЗУ |
| `display`  | Показать содержимое ОЗУ на экране (через GPU) |
| `exit`     | Завершить работу программы |
