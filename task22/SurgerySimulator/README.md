# Симулятор хирургической операции

Программа имитирует действия хирурга, который через ассистента запрашивает инструменты и выполняет операцию.  

## Команды

| Команда    | Аргументы                                      | Действие                                                       |
|------------|------------------------------------------------|----------------------------------------------------------------|
| `scalpel`  | X1 Y1 X2 Y2 (две точки)                        | Выполняет разрез между точками (X1,Y1) и (X2,Y2).              |
| `hemostat` | X Y (одна точка)                               | Накладывает зажим в указанной точке.                           |
| `tweezers` | X Y (одна точка)                               | Применяет пинцет в указанной точке.                            |
| `suture`   | X1 Y1 X2 Y2 (две точки)                        | Делает шов между точками. Если координаты совпадают с начальным разрезом, операция завершается. |

Операция всегда начинается с команды `scalpel` и заканчивается командой `suture` с теми же координатами, что и в начале.
